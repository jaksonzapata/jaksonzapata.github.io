Project Description

Built a real-time video chat web application using Next.js, React, and WebRTC, enabling peer-to-peer video communication between anonymous users. This project demonstrates the implementation of modern web technologies to create seamless real-time communication experiences.

Implemented media streaming and session management using the MediaDevices and Permissions APIs to access and monitor camera and microphone usage. The application handles all aspects of video call setup, connection management, and graceful disconnection.

Integrated Socket.IO to handle real-time messaging and WebRTC signaling for session creation. This enables instant connection establishment between users and maintains stable communication channels throughout the video chat session.

Designed a responsive, user-friendly interface with CSS Modules, including features like fullscreen toggle, chat auto-scroll, and skip/start controls. The interface adapts seamlessly across different devices and screen sizes while maintaining optimal performance.
